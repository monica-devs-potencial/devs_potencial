# Mario_bros

Cópia do repositório [devs_potencial](https://github.com/daniel-devs-potencial/devs_potencial) com as mesmas imagens e vídeo de fundo.

## Estrutura

- `index.php` / `index2.php` — Páginas públicas principais
- `servicos.php` — Página de serviços
- `style.css` — Estilos do site
- `script.js` / `script1.js` — Scripts JavaScript
- `video.mp4` — Vídeo de fundo
- `ico-imagem-fundo.png` — Imagem de fundo
- `logo.png` / `logo-header.png` — Logos
- `mario.png` — Imagem decorativa
- `admin/` — Painel administrativo
- `api/` — APIs REST (auth, contact, pix, plans, services, settings)
- `app/` — Classes PHP (db, mailer)
- `database/` — Schema SQL
- `config.example.php` — Exemplo de configuração (copiar para `config.php`)
- `composer.json` — Dependências PHP
